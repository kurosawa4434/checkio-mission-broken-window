Please restore the broken window glass.

You are given a list of lists as input values. Each list of integer represents a piece. It always has a horizontal base, which forms the upper or lower side of the window. Each integer represents the height from the base and is a value measured at equal intervals from the left edge. Therefore, the length of the list represents width.

The answer should be returned in the two lists. (upper list, lower list in order)

  • In the output, each piece is represented in the order given as input. (starting with 0)
  • The bases of the pieces in the same list must always be adjacent to each other.
    • (In other words, one piece doesn't form both the top and bottom of the window.)
  • It is not necessary to consider turning over piece.

{% endif %}
